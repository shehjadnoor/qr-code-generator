# QR Code Generator

A clean and modern QR Code Generator built using HTML, CSS, and JavaScript.  
Users can enter any text or URL, generate a QR code instantly, and download it as an image.

## Features
- Generate QR codes for text or links
- Download QR code as a PNG image
- Simple and modern user interface
- Responsive layout
- Personal branding included

## Technologies Used
- HTML
- CSS
- JavaScript
- QRCode.js library

## How It Works
1. Enter text or a URL in the input field
2. Click **Generate QR**
3. The QR code appears instantly
4. Click **Download QR** to save the image
